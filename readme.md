## Run Locally

- Download & install Go https://go.dev/doc/install

- Clone the project

```bash
  git clone https://github.com/ridho21/belajar-golang.git
```

- Go to the project directory

```bash
  cd belajar-golang
```

- Open the folder 

```bash
  cd slice
``` 

- Run the main program

```bash
  go run main.go
```
